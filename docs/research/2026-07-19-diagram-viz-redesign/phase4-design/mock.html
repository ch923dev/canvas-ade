Direct-manipulation editing for engine:'expanse' diagram cards — drag, re-route, palette, inline labels, and node→agent comments — without leaving the calm-UI contract. Static render stays the default; the editor mounts only on focus. Built with the live index.css tokens; dark-only, matching the app.

03

Palette — closed vocabulary only

Kind (shape), status (color), and an optional icon from the host Icon.tsx registry. No freeform colors or shapes — the calm contract, enforced.

KKind → silhouette + line icon. step · decision (octagon) · data · service · artifact · actor (pill) · note (dashed). Drives shape; never color.
SStatus → the one palette. neutral · active (accent) · done (ok) · warn · error · muted. Wash + half-border + glyph — color is never the sole carrier.
IIcon → host registry. Search Icon.tsx; unknown name renders none. Agents & users can pick from the closed set, never raw hex.
Guardrail The palette exposes exactly the enums MAIN already validates. A dropped node is a normal upsertNode spec op — same caps, same confirm path an agent's write takes.
