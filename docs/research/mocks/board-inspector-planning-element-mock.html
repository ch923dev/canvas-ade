Placement (Decision 4): the ELEMENT section sits at the top, under the identity head and above the always-present Tools/Canvas — the selection is the current focus, so its controls lead (Figma's right panel does the same). When the selection clears, the section unmounts and the panel returns to Tools-first. The board is never deselected by picking a tool (P3 rule), so the panel stays revealed throughout. Empty state (no element selected) = today's P3 panel exactly: head · Tools · Canvas · Duplicate-board foot, no Element section. Note the two Duplicates are disambiguated: element Duplicate (in the section) vs Duplicate board (foot).
