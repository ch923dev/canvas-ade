Mechanics in the real app: the OUT frame is the switch-away snapshot we already capture for the dock thumbnails, shown as an overlay above the unmounting canvas — so the animation runs while save/park/load happen underneath, adding zero wait. A load error drops the overlay instantly (error screen stays reachable); a watchdog clears it if a load hangs. Total added perception: ~0ms when the load is slower than 500ms, ~250ms of pure polish when the load is instant.
