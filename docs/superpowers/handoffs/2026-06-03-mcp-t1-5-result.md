## What shipped

A read-only, STRUCTURED last-result resource — a verdict + summary + references, NOT raw logs
(raw scrollback is T1.4's `canvas://board/{id}/output`). v1 is an **observational shell**: there is
no writer until M4's `write_result` tool, so every board returns `{ present: false }`. The shape is
designed so M4 fills the optional fields without a contract change.

  `MockOrchestrator` returns `{ present: false }`.
